# Short Link Service

基于 Fastify + Vercel + Supabase 实现的短链接服务。

## 在线访问

访问地址： [https://short.pangcy.cn](https://short.pangcy.cn)

## Running Locallly

```bash
npm i
npm i -g vercel@latest
vercel link
vercel env pull .env.development.local
vercel dev
```
